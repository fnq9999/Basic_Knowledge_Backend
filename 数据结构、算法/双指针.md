双指针
===========
### 连续数组的问题<br>
     固定j，i满足情况的条件下，{[0,i-1]，j}都不满足。


### leetcode 11 盛最多水的容器<br>
     
```c++
class Solution {
public:
    
    int maxArea(vector<int>& height) {
        int ans=0;
        int l=0,r=height.size()-1;
        while(l<r){
            ans=max(ans,min(height[l],height[r])*(r-l));
            if(height[l]<height[r]){
                l+=1;
            }else{
                r-=1;
            }
        }
        return ans;
    }
};
```
